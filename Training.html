@author : Richard Delwin Myloth, Pavan Rajkumar

Training.Git_Cmds.get_bugfix_commits()¶

This function is used to obtain all the commit id which have BUG or FIX in their commit messages which indicate that, that particular commit was used to fix a bug which was identified.

Raises ValueError if not buggy commits are found. This may be due to the fact that the repository might be a small one, or the commit messages are not approprately labelled as BUG or FIX

Returns

List of commit ids which fixed a bug

Training.Git_Cmds.get_latest_commit()¶

To get the latest commit, this commit is not used since it is used in the python script located in .git/git_hooks directory to execute this.

Returns

git output of the latest commit details

Training.Git_Cmds.link_fixes_to_bugs(fix_commits)¶

This function is used to find those commits responsible for introducing bugs into the software

Parameters
